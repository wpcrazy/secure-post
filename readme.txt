== Description ==

This is a very simple post secure plugin, which lets you secure your post from not logged in user. You can choose your desire category to lock the post.

== Installation ==

1. Upload `secure-post.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
